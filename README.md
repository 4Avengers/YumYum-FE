**나의 맛집 리스트를 공유하고, 지인이 추천하는 맛집을 저장하는 ‘맛집 공유 SNS’ 입니다.**

지도에서 한눈에 쉽게  내 주변 맛집을 보고 친구의 추천 맛집 리스트를 볼 수 있습니다.
